# Automated-Cell-Counting-in-Histology-Images
Repository for the Dissertation "Automated Cell Counting in Histology Images".

The libraries used can be found in `env.yml` - this file can be used to create an environment using conda, alternatively install the required modules in the correct versions (if listed)
Watershed and CPN's files can be run normally, using a notebook. 

However, `gt_extraction_streamlit.py` must be run using a terminal.
Direct the terminal to the folder you have the file in, then use the command:

`streamlit run gt_extraction_streamlit.py`
